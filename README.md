# lucerrne.github.io
This is my personal page which is deployed on GitHub Pages. 
I enjoy photography, so I use it to upload my photography photos. I utilized a waterfall layout and integrated with the Google API, allowing for the generation of a beautiful text description after uploading an image.

这是我在github上的第一个项目，是部署在github.io上的个人网站，我喜欢摄影所以用来上传摄影照片，我使用了瀑布流排版以及接入google api接口，可以在上传图片后生成一段精美的文字描述.
